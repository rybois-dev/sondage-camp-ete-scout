Sondage du camp d'été

Vos données :

Quelles données sont collectées ?

  • Le nom de votre enfant.
  • Le prénom de votre enfant.
  • Le folklore souhaité pour le camp.
  • Le nombre de jours d'exploration que ce dernier souhaite pour son camp.
  • Les activités sportives et autres types d'activité qu'il souhaite pour son camp.
  • Le nom et prénom de 3 personnes qu'ils souhaitent avoir dans leur sizaine.
  • Si votre enfant a un régime alimentaire particulier (à préciser aux responsables en dehors de ce sondage).
  • Si votre enfant suit un traitement pour le camp (les détails des médicaments à prendre est à communiquer aux responsables en dehors de ce sondage).
  • Le folklore à l'année que votre enfant souhaite pour l'année prochaine.
  • Un feedback sur l'année.

Comment sont utilisées vos données ?

Toutes les informations concernant votre enfant autre que son nom et prénom, ne sont pas stockées dans la base de données du site.

Ces informations sont envoyées par mail à l'adresse mail bccalvisson@gmail.com via le service EmailJS.

Les données saisies sont conservés sur la boite mail bccalvisson@gmail.com jusqu'à la fin de l'année scout de votre enfant.

Aucune information n'est stockée par le service EmailJS.

Vos droits en matière de protection des données

Conformément au RGPD, vous disposez des droits suivants concernant vos données personnelles :

  • Droit d'accès : Vous pouvez demander à accéder à vos données personnelles.
  • Droit de rectification : Vous pouvez demander la correction de vos données personnelles si elles sont inexactes ou incomplètes.
  • Droit à l'effacement : Vous pouvez demander la suppression de vos données personnelles sous certaines conditions.
  • Droit à la limitation du traitement : Vous pouvez demander la limitation du traitement de vos données personnelles sous certaines conditions.
  • Droit à la portabilité des données : Vous pouvez demander à recevoir vos données personnelles dans un format structuré, couramment utilisé et lisible par machine.
  • Droit d'opposition : Vous pouvez vous opposer au traitement de vos données personnelles pour des raisons spécifiques.
  • Droit de ne pas faire l'objet d'une décision individuelle automatisée : Vous avez le droit de ne pas être soumis à une décision basée uniquement sur un traitement automatisé, y compris le profilage, qui produit des effets juridiques vous concernant ou vous affecte de manière significative.

Pour exercer ces droits, veuillez contacter bccalvisson@gmail.com.
